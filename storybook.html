NSM Dark Theme Storybook

Version 2.0.0 rebases the design system onto the NSM (Network Security Manager) palette and draws in the component definitions that were previously missing. Every token and component is rendered live and mapped to its CSS variable for the dev team.

Collapse note. To keep the token set lean, NSM's warning-yellow and link/info-blue were folded onto the orange accent. The LED dot and status flag below therefore show yellow/blue states as orange. Add severity.warning-yellow / severity.info tokens later if those distinctions are required.
